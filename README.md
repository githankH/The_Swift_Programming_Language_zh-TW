我們非常需要大家一起加入協助翻譯工作，加入協助翻譯的方式下列兩種方式都可以：

- Fork 本專案，修正後以 Pull Request 方式提出修正，請自行在 README.md 感謝人員中加入自己的名字
- 加入 CocoaHeads Kaohsiung 翻譯計畫，請在 Issue 中提出「我想加入本計畫」我們就會把該帳號加入翻譯小組名單，您就可以直接參與修改
